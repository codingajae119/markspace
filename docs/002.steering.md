@docs/projects.md를 기준으로 /kiro-steering을 실행해 product/tech/structure steering을 만들어줘.
아래 규칙을 tech/structure steering에 반드시 포함해줘:
- 런타임/패키지 관리는 uv. 모든 실행은 uv run 기준.
- 설정 단일화: backend의 경우 모든 모듈이 동일한 config.yml(비밀 아닌 설정)과 .env(API key 등 secret만)만 사용. 로더는 pydantic-settings로 통일. frontend도 단일 설정 파일 적용
- spec 산출물은 한국어로.
